## Bootstrap blog

- Install Jekyll and Grunt dependency

    ```
    $ sudo gem install jekyll 
    $ npm install
    ```
    
- Go to <BLOG_HOME>/ and run Grunt to optimize image/css/js

    ```
    $ grunt
    ```

- Run blog

    ```
    $ jekyll server -w
    ```

> Cheers!
