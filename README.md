## Bootstrap blog

- Install dependencies

    ```
    $ bundle install 
    $ npm install
    $ npm install -g grunt-cli
    ```
    
- Go to <BLOG_HOME>/ and run Grunt to optimize image/css/js

    ```
    $ grunt
    ```

- Run blog

    ```
    $ jekyll server -w
    ```
